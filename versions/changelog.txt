# RPG Mod — Список изменений


## [v2.1.0]
- Внедрены новые характеристики: Сила, Ловкость, Живучесть, Интеллект и Маг. Сила. Убраны старые.
- Добавлена система Защиты (Defense), снижающая получаемый урон на процент, зависящий от Живучесть.
- Итоговые статы вынесены в отдельную новую вкладку «Билд» (Build) в меню прокачки.
- Проведено масштабное ООП-рефакторирование способностей (исправлен расчет скейлингов урона и кулдаунов в зависимости от статов игрока).
- Обновлены и актуализированы внутриигровые гайды.

## [v2.0.0]
- Добавлен HUD кулдаунов и стамины в левом нижнем углу экрана
- Полностью переработано меню "Настройки" для мода и радиальное дерево Магии
- Доработана локализация для всех физических и магических навыков
- Изменен подход к координатам хитбокса в UI для корректного масштабирования и кликов
- Появились характеристики Урона / Силы / Лечения, отображающиеся в меню прокачки
- Переработано Древо Магии: базовые заклинания — корни древа (Tier 0)
- Создан гайд по игровому циклу (`Guides/guide.txt`)

## [v1.3.0]
- ООП-рефакторинг: `AbstractAbility` с generic Builder, объединяющий `Ability` и `MagicAbility`
- Единый `AbilityRegistry` (объединил `AbilityRegistry` + `MagicSkillRegistry`)
- Объединены сетевые пакеты: `USE_ABILITY` + `USE_MAGIC_SKILL` → один пакет
- Единая карта `skillLevels` в `PlayerStatsData` с миграцией NBT
- Обновлены все экраны и HUD для единого API
- Удалён `MagicSkillRegistry.java`

## [v1.2.0]
- Полный редизайн UI: стекломорфизм, анимации, карусель карт Таро для выбора стихии
- Создан интерфейс `IAbility` для стандартизации всех способностей
- Кулдауны корректно работают для всех магических заклинаний в HUD

## [v1.1.0]
- HUD: анимация перезарядки (cooldown) — затемнение иконки сверху вниз + таймер
- Настройки: исправлен скролл, корректные кнопки биндинга клавиш

## [v1.0.0]
- Изменена архитектура: характеристики (урон, мана, стамина, кулдаун) теперь вынесены в отдельные классы (например `HealStat`, `MaxManaStat`) в пакете `stats`.
- Добавлена новая команда `/rpg <add/remove/set> <игрок> <характеристика> <значение>` для управления индивидуальными характеристиками (работает с PlayerStatsData).
- Система автообновления версий и генерации changelog
- Двуязычная поддержка (EN/RU) для `/rpg help`
- Переработка `/rpg reset`: разделение по игрокам, очистка клиентских клавиш через S2C пакеты
- Исправлен биндинг клавиш в настройках (включая кнопки мыши)
