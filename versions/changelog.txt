# RPG Mod Changelog

## v1.0.0
- Initialized automated version tracking and changelog generation.
- Implemented dual-language (EN/RU) support for `/rpg help`.
- Overhauled `/rpg reset` to parse specific player targets and forcibly clear client UI keybinds via S2C packets.
- Fixed keybind assignment bugs inside `SettingsScreen` (allowed rebinding to mouse buttons).
- Re-architected Magic Trees: Removed 'Mastery' node dependency and set base elemental spells as Tree Roots (Tier 0).
- Created a comprehensive core-loop guide locating inside `Guides/mod_guide.txt`.
- Added the `github-auto-commit` and `skill-creator` agent automation skills.
