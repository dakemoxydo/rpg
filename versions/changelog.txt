# RPG Mod Changelog

## v1.0.0
- Добавлена система автоматического обновления версий и генерации списка изменений.
- Реализована поддержка двух языков (EN/RU) для команды `/rpg help`.
- Переработана команда `/rpg reset`: добавлено разделение по игрокам и принудительная очистка пользовательских клавиш клиента через S2C пакеты.
- Исправлены баги с назначением клавиш в `SettingsScreen` (теперь можно переназначать на кнопки мыши).
- Переработано Древо Магии: удалена зависимость от ноды 'Mastery', базовые стихийные заклинания стали корнями древа (Tier 0).
- Создан подробный гайд по основному циклу игры, расположенный в `Guides/mod_guide.txt`.
- Добавлены навыки автоматизации агента `github-auto-commit` и `skill-creator`.


## v1.1.0.0
- В интерфейс (HUD) добавлена анимация перезарядки (cooldown), затемняющая иконку навыка сверху вниз с отображением таймера в секундах.
- Исправлен интерфейс Настроек: починен скролл и корректно отображаются кнопки для назначения горячих клавиш выученных навыков.
